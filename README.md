# 星际战区-酒馆数据采集器

最新版下载地址：https://sc-cdn.yuanfen.net/downloads/collector-latest.zip

## 采集原理
星际酒馆会将比赛次数、积分等信息存储至本地，本工具从本地保存的文件中读取数据并上报至服务器。

## 采集频率
首次打开会主动上传，后续在检测到文件变化后会自动上传。若日志中看到错误，可过段时间点击手动上传。
